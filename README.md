# Website Pendaftaran Event Kampus

## Deskripsi Project
Website ini digunakan untuk pendaftaran peserta seminar teknologi kampus.
Peserta dapat mendaftar secara online melalui form yang tersedia.

## Fitur Utama
- Landing page informasi event
- Form pendaftaran peserta
- Validasi input menggunakan JavaScript
- Penyimpanan data menggunakan localStorage
- Pembatasan kuota maksimal 50 peserta
- Menampilkan data peserta tanpa reload halaman
- Fitur hapus peserta
- Tampilan responsif menggunakan CSS Flexbox

## Teknologi yang Digunakan
- HTML
- CSS (Flexbox)
- JavaScript (DOM & localStorage)

## Alur Logika Sistem
1. Website memuat data peserta dari localStorage.
2. Peserta mengisi form pendaftaran.
3. Sistem melakukan validasi data.
4. Jika valid dan kuota tersedia, data disimpan.
5. Data ditampilkan langsung pada halaman.
6. Peserta dapat dihapus dari daftar.
7. Jumlah peserta diperbarui secara otomatis.

## Cara Menjalankan Project
1. Buka file index.html di browser.
2. Isi form pendaftaran.
3. Data akan tersimpan otomatis di browser.
